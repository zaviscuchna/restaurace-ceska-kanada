# Restaurace Česká Kanada — Zvůle

Platí studio standardy z `../CLAUDE.md` (KSH umělecká vize). Tohle jsou specifika projektu.

## Co to je
Web pro **starší restauraci v autokempu u rybníka Zvůle** (Kunžak, region Česká Kanada).
Charakter místa: klid, borový les, voda, nostalgie kempu, výletní/rekreační místo.
Klient/autor: Záviš. Cíl: web s atmosférou, ručně dolaďovaný. **NE AI slop, NE template.**

## Stack
Next.js 16 (App Router) + React 19 + Tailwind v4 + TS. ⚠️ Next 16 má breaking changes — viz `AGENTS.md`.
